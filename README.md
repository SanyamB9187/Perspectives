📌 Project Title

Perspectives – Multi-Category Blogging Platform

🚀 Overview

Perspectives is a dynamic web application that presents curated content across multiple domains, including cinema, sports, and books. The project focuses on building a clean UI, efficient rendering of content, and a smooth reading experience using core web technologies.

🛠️ Tech Stack

Frontend

HTML5
CSS3
JavaScript (Vanilla JS)

✨ Features

Cinema section with dynamic content display
Sports section with structured content layout
Books section for curated entries
Dynamic image rendering
Blog-style content presentation
Clean and readable UI
Responsive layout
